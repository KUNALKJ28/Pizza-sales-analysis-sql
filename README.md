# 🍕 Pizza Sales Analysis using MySQL

## 📌 Project Overview

This project focuses on analyzing pizza sales data using MySQL and SQL queries to derive meaningful business insights. The project demonstrates how SQL can be used for data analysis by examining customer ordering patterns, revenue generation, and the performance of different pizza categories.

## 🎯 Objectives

* Analyze pizza sales data using SQL queries.
* Calculate total revenue generated from pizza sales.
* Identify the top-selling pizzas and categories.
* Determine peak ordering hours and customer trends.
* Generate business insights to support decision-making.

## 🛠️ Tools and Technologies Used

* MySQL Workbench
* MySQL Server
* SQL (Structured Query Language)
* Microsoft PowerPoint (Documentation and Presentation)

## 🗄️ Database Structure

The project consists of four relational tables:

1. **Orders**

   * Stores order ID, order date, and order time.

2. **Orders_Details**

   * Contains pizza ID, order ID, and quantity of pizzas ordered.

3. **Pizzas**

   * Stores pizza ID, size, and price details.

4. **Pizza_Types**

   * Contains pizza names, categories, and ingredients.

## 📊 Analysis Performed

* Total number of orders placed
* Total revenue generated
* Highest-priced pizza
* Most common pizza size ordered
* Top 5 most ordered pizza types
* Category-wise distribution of pizzas sold
* Revenue contribution by each pizza category
* Peak ordering hours and customer ordering patterns

## 🔍 Key SQL Concepts Used

* SELECT Statements
* Aggregate Functions (SUM, COUNT, AVG)
* GROUP BY
* ORDER BY
* LIMIT
* INNER JOIN
* Subqueries
* Aliases

## 📈 Project Outcome

This project helped in understanding how SQL can be used to transform raw sales data into meaningful insights. The analysis identified customer preferences, popular pizza categories, revenue trends, and peak business hours, demonstrating the practical application of SQL in business analytics and decision-making.
